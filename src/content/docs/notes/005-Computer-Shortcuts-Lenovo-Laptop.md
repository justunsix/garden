## Trackpoint (Red dot)

| Description                    | Shortcut                               |
|--------------------------------|----------------------------------------|
| Move                           | Pointing Stick                         |
| Select, open item (left click) | Left button                            |
| Shortcut menu                  | Right button                           |
| Scroll                         | Hold middle button, use pointing stick |

## Trackpad

| Description | Shortcut |
|----|----|
| Select, open item (left click) | Tap area except bottom right |
| Shortcut menu | Tap bottom lower right area or Two finger tap |
| Scroll | Two finger hold, scroll |
| Zoom | Two finger pinch or expand |
| Task view, restore apps | Swipe three fingers up |
| Show desktop | Swipe three fingers down |

Applications may limit trackpad functionality
