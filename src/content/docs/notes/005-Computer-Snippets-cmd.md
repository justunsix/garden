``` shell

# Set environment variable
setx http_proxy "1.1.1.1:3039"

# Shutdown
## Shutdown immediately
shutdown /p
## Shutdown computer gracefully after 1 minute
shutdown -s
## Shutdown given a time in seconds
shutdown -s -t 30

```
