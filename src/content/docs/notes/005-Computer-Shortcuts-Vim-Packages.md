## vim-dabod-ui - navigation through databases and allows saving queries

| Description              | Shortcut                              |
|--------------------------|---------------------------------------|
| Buffer, delete           | d                                     |
| Buffer, redraw           | R                                     |
| Database, new            | :DBUIAddConnection or A (dbui drawer) |
| Database, show UI        | :DBUI                                 |
| Database, toggle details | H                                     |
| Help                     | ?                                     |
| Query, run               | :w                                    |
| View, vertical split     | S                                     |
