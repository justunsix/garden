Source: [Notebook Basics — Jupyter Notebook
documentation](https://jupyter-notebook.readthedocs.io/en/latest/examples/Notebook/Notebook%20Basics.html)
and `jupyter notebook` Help \> Keyboard Shortcuts

| Name                     | Shortcuts        |
|--------------------------|------------------|
| Cell, Create             | a, b             |
| Command Palette          | Ctrl + Shift + c |
| Continue                 | F9               |
| Fullscreen               | F11              |
| Help, Shortcuts          | Ctrl + Shift + h |
| Navigation, cell         | up/k, down/j     |
| Navigation, command mode | Esc              |
| Navigation, edit cell    | Enter            |
| Navigation, run          | Shift + enter    |
| Save                     | Ctrl + s         |
| Undo                     | Ctrl + z         |
