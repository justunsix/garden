| Name | Shortcuts |
|----|----|
| Navigation | Arrow keys or Ctrl + p / n or Ctrl + k / j |
| Navigation, Scroll pane up or down | Page Up or Page Down |
| Select entry | Enter |
| Selected entry, copy | Ctrl + y |
| Remote control mode, toggle | Ctrl + t |
| Help | Ctrl + h |
| Preview panel, toggle | Ctrl + o |
| Quit | Esc |
| Shell integration, Shell history | Ctrl + r |
| Shell integration, autocomplete for current command | Ctrl + t |
