Ventoy is an open source tool to create bootable USB drivers with
operating system files like ISO, IMG, and other types of images. See
features and supported file types at
[Ventoy](https://www.ventoy.net/en/index.html) and documentation at
[Ventoy - Get Started](https://www.ventoy.net/en/doc_start.html)

## Use Cases for using Ventoy

- Create a USB to boot or install an operating system
- Use a menu system to boot into multiple operating systems and browse
  files
- Add/update systems as needed as files instead of creating new USB
  every time
- Use features like linux persistence, automated Installation, and use
  USB regularly

Good for:

- System administrators
- People who fix computers or recover systems
- Trying or booting into multiple different operating systems
