List of items to bring on day or multi day hike to Glen Rouge

- Map + cover + watch / compass
- Headlight + spare batteries
- Sunglasses + screen
- Clothing + compression sacks x 3
  - Fleece jacket (Columbia)
  - Fleece pants?
  - Rain pants
  - Hat
  - Bandana x 2
  - Wool socks x 2
  - Underwear - silk
  - Gloves (brown + overhand)
  - Heavier sweatshirts
  - Red raincoat
- First aid kit small
  - Blisters
  - Adhesives
  - Tape
  - Gloves + CPR kit
  - Disinfecting chemical
  - Towelettes
- Matches, alcohol, stove
- Kit:
  - Rescue Tool
  - Duct tape
  - Floss
- Food (see listing)
- Water bottle (Nalgene, Silicon)
- Garbage bag, space blanket
- Storm whistle
- Walking stick
- Sleeping bag + pad (Exercise mat)
- Spork + Silicon bowl
- Pack towel
- Tent
- Book(s)
- Toiletries (small toothbrush, small toothpaste, small soap)
- Backpack
- Pad + pen
- Cash + ID
