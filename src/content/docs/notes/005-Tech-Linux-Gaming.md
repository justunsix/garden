## Linux Gaming How To

Source: [THE GAMING ON LINUX GUIDE: How to play anything: Steam, Epic,
Ubisoft, Origin, Battle.net, GoG - The Linux Experiment -
YouTube](https://www.youtube.com/watch?v=v9tb1gTTbJE), [Linux Gaming in
2026 - Learn Linux TV -
YouTube](https://www.youtube.com/watch?v=qsSIxyFcS0M)

Different options are available.

1.  Use [Steam](https://store.steampowered.com/) with Proton
    Compatibility tool based on Wine

2.  Use [Heroic Games Launcher](https://heroicgameslauncher.com/)
    (flatpak) for:

    - Epic Games
      - Set appropriate Wine/proton options
    - GOG

3.  Use [Lutris](https://lutris.net/) (flatpak) for:

    - Origin
    - Ubisoft
    - Steam
    - GOG
    - Battle.net (install inside Lutris, use Battle.net, should run
      fine)
    - Other stores
    - For Lutris: select appropriate scripts/versions for a game, should
      be best for game

Apply FSR (requires GE Proton) to set game to lower resolution and
upscale to your monitor's resolution

- For Steam, set in launch options
- For Heroic Launcher, enable FSR hack
- For Lutris, set in game options

Heroic and Lutris can use Wine with additions for Windows games.

1.  Use [Crossover](https://www.codeweavers.com/crossover) which uses a
    proprietary core with Wine. Search [Crossover compatibility
    database](https://www.codeweavers.com/compatibility/) for software
    you want to run.
