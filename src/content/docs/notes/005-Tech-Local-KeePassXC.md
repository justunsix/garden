## Command Line Interface (CLI)

``` bash

# Open database with key file
keepassxc-cli open -k /path/to/keyfile /path/to/database.kdbx

# Get help while inside database
help

# Search for entry
search <search term>

# Get entry password and copy to clipboard
clip <entry name>

```
