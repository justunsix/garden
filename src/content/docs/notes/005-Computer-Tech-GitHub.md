## Website Shortcuts

Command Palette = ctrl + k

## GitHub.dev

Open any repository on GitHub.com with GitHub.dev, a web editor. If
using Firefox, turn off tracking protection on Github.dev per [VS Code
Issue \#234152](https://github.com/microsoft/vscode/issues/234152),
otherwise you may get an `vfs` error.
