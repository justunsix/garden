Org Citations for oc aka Org Citations Library (Emacs built in package)
and [citar](https://github.com/emacs-citar/citar).

| Name                       | Shortcut               |
|----------------------------|------------------------|
| Citation, insert           | org-cite-insert        |
| Citation, edit             | citar-insert-edit      |
| Open citation              | citar-dwim, embark-act |
| Open citation source       | citar-open-source      |
| Open citation source, link | citar-open-links       |
