## About

Works on Android, BSD, Linux, macOS, Windows.

Can be used on multiple shells including bash, fish, Xonsh, zsh.

Install using [package managers and install per
shell](https://starship.rs/guide/#%F0%9F%9A%80-installation).

## Configuration

- Default configuration file is at `~/.config/starship.toml`. It can be
  edited using environment variables
- Default Starship log warnings are at `~/.cache/starship/...log`

## Help

- Explain current prompt symbols and text: `starship explain`
