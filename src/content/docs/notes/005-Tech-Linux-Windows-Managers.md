## Things to use with Windows Managers

- Application launcher:
  - Dmenu: Opens applications in `$PATH`
  - Rofi
- Ideas of good modules:
  - [GitHub - AtifChy/xmonad: my Xmonad
    configuration](https://github.com/AtifChy/xmonad#preview)
- Night Light:
  - [gammastep](https://gitlab.com/chinstrap/gammastep):
    `gammastep -P -O 2000` to set the temperature to 2000K
  - redshift: `redshift -t 2700:2700 -m randr -l 43:79 &`
- Clipboard manager:
  - [greenclip, integrated with
    rofi](https://github.com/erebe/greenclip)
  - Coppq
- Compositor (required for some applications and windows effects:
  - picom
- Key ring manager:
  - Gnome Keyring
- polkit agent - handle GUI password prompts:
  - xfce-polkit
  - polkit-gnome
