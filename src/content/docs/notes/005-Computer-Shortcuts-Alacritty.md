| Name                                       | Shortcuts                    |
|--------------------------------------------|------------------------------|
| Copy                                       | Mouse over text to be copied |
| Full screen                                | Alt + Enter                  |
| Open (file, URL)                           | Enter, Mouse 1               |
| Search                                     | Ctrl + Shift + f             |
| Vi Mode, copy                              | y                            |
| Vi Mode, scroll back                       | Ctrl + Shift + Space         |
| Vi Mode, search backward                   | ?                            |
| Vi Mode, search forward                    | /                            |
| Vi Mode, select                            | v                            |
| Vi Mode, select mode semantic, line, block | Alt + v, Shift + v, Ctrl + v |
