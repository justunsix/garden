## Creation Prompts and Workflows

Source: [Accelerate Your Writing Workflow with AI Tools & Prompts -
YouTube](https://www.youtube.com/watch?v=D5XIsnDyrGY)

### Prompt Types

| Prompt type | Description | Example |
|----|----|----|
| Zero shot | Simple prompt and use model's own training | Writing a 30 second script about AI |
| Few shot | Give context, like output examples | Write script in this format and tone |
| Chain of thought | Reason in steps | Run through these steps to research |
| Take a deep breath | Slow down and examine technical details | Go through these technical details |
| Brainstorm then select | Provide ideas, then find best ideas | Give 10 ideas, iterate to fewer ideas |
| Role play | Ask model to assume a persona | Act like a product manager, outline this feature |

### Workflow for creation

#### 1. Brainstorm

- Audience
- Gaps to address
- Indicators of success

Not done by AI: You determine strategy, pick option or fact check.

#### 2. Outline

- Key points
- Content creation
- Steps to create

Not done by AI: You ensure alignment with your organization and
priorities

#### 3. Draft

- Create initial draft, headings
- Style, tone

Not done by AI: You choose branding, data to choose

#### 4. Revise

- Improve clarity, grammar, spelling, find errors
- Give feedback based on audience persona

Not done by AI: Ensure content is correct, human tone, and finalize
