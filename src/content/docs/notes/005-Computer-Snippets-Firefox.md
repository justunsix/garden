Firefox command line options \[fn:1\].

``` shell

# Start Firefox profile manager and select profile
firefox -ProfileManager

# Set Firefox as default browser
firefox -setDefaultBrowser

# Open URL in a new ta[[https://wiki.mozilla.org/Firefox/CommandLineOptions#Command_line_options][Firefox/CommandLineOptions - MozillaWiki]]b
firefox -new-tab URL

```
